Data Cleansing Functions

2020-04-24
data cleansing, data preprocessing, pandas, python, set
  • лҚ°мқҙн„° л¶„м„қм—җ м•һм„ң мһҳлӘ»лҗң л¶Җл¶„мқ„ м°ҫм•„лӮҙкі  мқҙлҘј л©”кҫёлҠ” мһ‘м—…мқ„ лҚ°мқҙн„° м •лҰ¬(data cleansing)мқҙлқјкі  н•©лӢҲлӢӨ.
  • к°ңлі„ лҚ°мқҙн„°мқҳ нғҖмһ…(type), нҒ¬кё°(length) л“ұмқҙ мһҳлӘ»лҗң кІҪмҡ°лҸ„ мһҲкі , к°ңлі„ лҚ°мқҙн„°лҠ” м •мғҒм Ғмқҙм§Җл§Ң мқҙкІғл“Өмқ„ н•©міӨлҚ”лӢҲ(merge) кІ°мёЎм№ҳк°Җ м–‘мӮ°лҗҳкұ°лӮҳ мӨ‘ліөлҗҳлҠ” кІҪмҡ°лҸ„ мһҲмҠөлӢҲлӢӨ.
  • л¶„м„қмқҙлӮҳ мҳҲмёЎлӘЁлҚё мғқм„ұмӢң л¬ём ңмқҳ мӣҗмқёмқҙ лҗҳкё° л•Ңл¬ём—җ мІҳмқҢм—җ м •лҰ¬лҘј н•  н•„мҡ”к°Җ мһҲмҠөлӢҲлӢӨ.

  • лҚ°мқҙн„° м •лҰ¬м—җ мӮ¬мҡ©н•  мҲҳ мһҲлҠ” н•ЁмҲҳлҘј лӘҮ к°ң мҶҢк°ңн•©лӢҲлӢӨ.

1. лҚ°мқҙн„° нғҖмһ… нҷ•мқё

  • н•ЁмҲҳм—җ лҚ°мқҙн„°лҘј л„Јмқ„ л•Ң, нҠ№м • нғҖмһ…л§Ңмқ„ л°ӣм•„л“Өм—¬м•ј н•  л•Ңк°Җ мһҲмҠөлӢҲлӢӨ.

  • нғҖмһ…мқҙ мһҳлӘ»лҗҳл©ҙ м–ҙл””м„ к°Җ м—җлҹ¬к°Җ лӮҳм§Җл§Ң, к№ҠмқҖ кіім—җм„ң м—җлҹ¬к°Җ лӮҳл©ҙ мӣҗмқёмқ„ м°ҫкё° нһҳл“ңлӢҲ лҚ°мқҙн„°лҘј л°ӣмһҗл§Ҳмһҗ нғҖмһ…мқ„ м җкІҖн•ҳлҠ” кІғмқҙ нҡЁкіјм Ғмһ…лӢҲлӢӨ.

  • мҳҲмӢң

  • мҳҲмғҒ нғҖмһ…кіј мӢӨм ңк°Җ лӢӨлҘј кІҪмҡ°

2. кІ°мёЎм№ҳ нҷ•мқё

github: missingno

  • кІ°мёЎм№ҳ кҙҖл Ён•ҳм—¬ missingnoлқјлҠ” мўӢмқҖ лқјмқҙлёҢлҹ¬лҰ¬к°Җ мһҲмҠөлӢҲлӢӨ.
  • м—¬кё°м—җ лҢҖн•ң кёҖмқҖ л§ҺмңјлӢҲ ліё кёҖм—җм„ңлҠ” лӢӨлЈЁм§Җ м•ҠкІ мҠөлӢҲлӢӨ.

2.1. numpy array

  • кІ°мёЎм№ҳк°Җ мһҲлҠ” numpy dataлҘј л§Ңл“Өкі , м—¬кё°м„ң кІ°мёЎм№ҳмқҳ к°ҜмҲҳмҷҖ мң„м№ҳлҘј нҷ•мқён•ҙ лҙ…мӢңлӢӨ.
  • кІ°мёЎм№ҳ index нҷ•мқё
  • кІ°мёЎм№ҳлҘј м ңмҷён•ң лҚ°мқҙн„° м¶”м¶ң

2.2. pandas DataFrame

  • jupyter notebookм—җм„ң cellмқҳ л§Ҳм§Җл§ү мӨ„м—җ dataframe мқҙлҰ„л§Ң мһ…л Ҙн•ҳл©ҙ мҳҲмҒң tableмқҙ лӮҳмҳӨм§Җл§Ң, cell мӨ‘к°„м—җ dataframeмқҳ лӮҙмҡ©мқ„ м•Ңм•„ліҙкё° мң„н•ҙ print(df)лҘј н•ҳл©ҙ мҲ«мһҗл“Өмқҳ лӮҳм—ҙлЎңл§Ң лӮҳмҷҖ м•Ңм•„ліҙкё° нһҳл“ӯлӢҲлӢӨ.
  • cell мӨ‘к°„м—җ dataframeмқ„ м¶ңл Ҙн•ҙлҸ„ мҳҲмҒҳкІҢ лӮҳмҳ¬ н•ЁмҲҳлҘј к°„лӢЁн•ҳкІҢ л§Ңл“Ө мҲҳ мһҲмҠөлӢҲлӢӨ.
  • 10н–ү x 3м—ҙ DataFrameмқ„ л§Ңл“Өм–ҙм„ң н…ҢмҠӨнҠён•©мӢңлӢӨ.
  • мқјл¶Җ лҚ°мқҙн„°лҘј лһңлҚӨн•ҳкІҢ кІ°мёЎм№ҳлЎң л§Ңл“ӯлӢҲлӢӨ.
  • .plot лӘ…л №мңјлЎң лҚ°мқҙн„°лҘј м¶ңл Ҙн•ҳл©ҙ кІ°мёЎм№ҳлҘј м ңмҷён•ҳкі  м¶ңл Ҙн•©лӢҲлӢӨ.
  • c='C', cmap='jet'лҘј м¶”к°Җн•ҙм„ң colormapк№Ңм§Җ мһ…нҳҖліҙкІ мҠөлӢҲлӢӨ.
  • мғүмғҒ кІ°мёЎм№ҳк№Ңм§Җ л°ҳмҳҒлҗҳм–ҙ лҚ°мқҙн„°к°Җ мӮ¬лқјм§‘лӢҲлӢӨ: (7, 7) м§Җм җмқҙ мӮ¬лқјмЎҢмҠөлӢҲлӢӨ.

2.3. кІ°мёЎм№ҳ н•©м§‘н•© м ңкұ°

  • м—җлҹ¬мқҳ мҶҢм§Җк°Җ лҗҳлҠ” кІ°мёЎм№ҳлҘј мһҗлҸҷмңјлЎң м ңкұ°н•ҙ мЈјм–ҙм„ң мўӢм§Җл§Ң, лҚ°мқҙн„° мҶҗмӢӨлҹүмқ„ м•Ң мҲҳ м—ҶмҠөлӢҲлӢӨ.
  • м•һм„ң мӮ¬мҡ©н•ң np.where()мҷҖ np.delete()лҘј мқҙмҡ©н•ҙм„ң кІ°мёЎм№ҳ мң„м№ҳлҘј нҷ•мқён•ҳкі  мӮӯм ңн•  мҲҳ мһҲм§Җл§Ң, л°©мӢ¬н•ҳл©ҙ м—үлҡұн•ң лҚ°мқҙн„°лҒјлҰ¬ л¶ҷм–ҙлІ„лҰҪлӢҲлӢӨ.
  • ліҙмӢңлӢӨмӢңн”ј л„Ө к°ң л°–м—җ м—Ҷм–ҙм•ј н•  м җмқҙ 7к°ңлӮҳ мһҲкі , y=xм—җм„ңлҸ„ лІ—м–ҙлӮ©лӢҲлӢӨ.
  • мқҙлҹҙ л•ҢлҠ” лҸ…лҰҪлҗң кІ°мёЎм№ҳлҘј лЁјм Җ н•©м§‘н•©мңјлЎң л§Ңл“  нӣ„м—җ x, y, cм—җм„ң лӘЁл‘җ мӮӯм ңн•ҙм•ј н•©лӢҲлӢӨ.
  • мқҙ кіјм •мқ„ н•ҳлӮҳмқҳ н•ЁмҲҳлЎң мӢӨн–үн•ҙ ліҙкІ мҠөлӢҲлӢӨ.
  • assertлҘј мӮ¬мҡ©н•ҙм„ң 'list', 'array', 'series'м—җл§Ң м Ғмҡ©лҗҳлҸ„лЎқ н–ҲмҠөлӢҲлӢӨ.
  • н•©м§‘н•©мқ„ кө¬н•ҳлҠ” set(A) | set(B) лӘ…л №мқ„ мӮ¬мҡ©н–Ҳмңјл©°,
  • *argsмҷҖ reduceлҘј мқҙмҡ©н•ҙм„ң мһ…л Ҙл°ӣлҠ” мқёмһҗмқҳ к°ҜмҲҳм—җ л¬ҙкҙҖн•ҳкІҢ лҸҷмһ‘н•©лӢҲлӢӨ.

  • м•„к№Ң л§Ңл“Өм—ҲлҚҳ, кІ°мёЎм№ҳк°Җ мһҲлҠ” numpy.ndarrayм—җ м Ғмҡ©н•ҙ ліҙкІ мҠөлӢҲлӢӨ.

  • к°Ғ н•ӯмқҳ кІ°мёЎм№ҳ к°ҜмҲҳмҷҖ н•Ёк»ҳ м „мІҙм ҒмңјлЎң лӘҮ к°ңмқҳ лҚ°мқҙн„°к°Җ л№ мЎҢлҠ”м§Җ м•Ңл ӨмӨҚлӢҲлӢӨ.

  • plotлҸ„ м •мғҒм ҒмңјлЎң м¶ңл Ҙлҗ©лӢҲлӢӨ.
    • matplotlibмқҖ мһҗмІҙм ҒмңјлЎң кІ°мёЎм№ҳлҘј м •мғҒм ҒмңјлЎң м ңкұ°н•ҳкі  plotмқ„ мҲҳн–үн•©лӢҲлӢӨ.
    • к·ёлҹ¬лӮҳ seabornмқҳ distplot()мқҖ кІ°мёЎм№ҳк°Җ мһҲмңјл©ҙ лҸҷмһ‘н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.
    • мӮ¬мӢӨ chk_nan()мқҖ plotліҙлӢӨ лӢӨлҘё мһ‘м—…мӢң кІ°мёЎм№ҳлҘј мҳӨлҘҳ м—Ҷмқҙ мІҳлҰ¬н•  л•Ң мң мҡ©н•©лӢҲлӢӨ.

3. лҚ°мқҙн„° мӨ‘ліөм№ҳ нҷ•мқё

pandas.DataFrame.duplicated
pandas.Seires.duplicated
pandas.Index.duplicated

  • pandasлҠ” мӨ‘ліөк°’мқ„ нҷ•мқён•  мҲҳ мһҲлҠ” .duplicated() м ңкұ°н•ҳлҠ” .drop_duplicates()лҘј м ңкіөн•©лӢҲлӢӨ.
  • мқҙмҷҖ кҙҖл Ёлҗң м„ӨлӘ…мқҖ мӣ№м—җ л§Һмқҙ мһҲмңјлӢҲ л‘җ к°Җм§Җл§Ң м „лӢ¬л“ңлҰ¬л Ө н•©лӢҲлӢӨ.

1. мӨ‘ліөм№ҳ нҷ•мқёмқҖ key к°’мқ„ л„Јкі  н•ҳмһҗ :

  • df.duplicated()мқҳ кё°ліёк°’мқҖ н–ү м „мІҙлҘј л№„көҗн•ҳлҜҖлЎң мқјл¶Җл§Ң мӨ‘ліөлҗҳл©ҙ лӢӨлҘё лҚ°мқҙн„°лЎң к°„мЈјн•©лӢҲлӢӨ.
    ліҙнҶө лҚ°мқҙн„°лҠ” кё°мӨҖмқҙ лҗҳлҠ” key к°’мқҙ мһҲмңјлҜҖлЎң .duplicated([{key_column}])лҘј мҲҳн–үн•ҳлҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ.
  • мӨ‘ліөм№ҳ к°ҜмҲҳлҠ” л’Өм—җ .sum()мқ„ л¶ҷмқҙл©ҙ к°„лӢЁн•©лӢҲлӢӨ.
    мӨ‘ліө м—¬л¶ҖлҘј True(1)/False(0)лЎң н‘ңкё°н•ң SeriesлҘј м¶ңл Ҙн•ҳкё° л•Ңл¬ёмһ…лӢҲлӢӨ.
  • мӨ‘ліөм№ҳлҘј м ңкұ°н•ҳкё° м „м—җ мӨ‘ліө лҚ°мқҙн„°лҘј лҪ‘м•„м„ң мңЎм•ҲмңјлЎң кІҖмӮ¬лҘј н•ҳлҠ” кІғмқҙ мўӢмҠөлӢҲлӢӨ.
    мӨ‘ліө мһҗмІҙк°Җ лӯ”к°Җ л§Һмқҙ мһҳлӘ»лҗң кІҒлӢҲлӢӨ. кІҪн—ҳмғҒ кІ°мёЎліҙлӢӨ мӨ‘ліөмқҙ мһҲмқ„ л•Ң кі мғқмҠӨлҹ° мқјмқҙ лҚ” л§Һм•ҳмҠөлӢҲлӢӨ.

2. .drop_duplicates()лҠ” кё°ліём ҒмңјлЎң мІ« лҚ°мқҙн„°лҘј лӮЁкёҙлӢӨ :

  • лҚ°мқҙн„°мқҳ мӨ‘ліөмқ„ нҷ•мқён•ң нӣ„м—җлҠ” м–ҙл–Ө кІғмқ„ м§Җмҡём§Җ м„ нғқн•ҙм•ј н•©лӢҲлӢӨ.
  • мӨ‘ліөм№ҳлҘј м ңкұ°н•ҳлҠ” лӘ…л №мңјлЎң drop_duplicates()лҘј мӮ¬мҡ©н•ҳлҠ”лҚ°, keep='first'к°Җ кё°ліё мҳөм…ҳмһ…лӢҲлӢӨ.
  • м—¬лҹ¬ мӨ‘ліө лҚ°мқҙн„° мӨ‘ мІ« лҚ°мқҙн„°л§Ңмқ„ лӮЁкё°кі  мӮӯм ңн•ҳкё° л•Ңл¬ём—җ мҳім§Җ лӘ»н•ң м„ нғқмқј мҲҳ мһҲмҠөлӢҲлӢӨ.
  • мӨ‘ліө лҚ°мқҙн„°мқҳ featureлҘј мў…н•©м ҒмңјлЎң мӮҙнҺҙліё нӣ„ м„ нғқн•ҙм•ј н•©лӢҲлӢӨ. л•ҢлЎңлҠ” мҲҳлҸҷ мӮӯм ңк°Җ н•„мҡ”н•©лӢҲлӢӨ.
  • м•„лһҳлҠ” м ңк°Җ мӢӨм ңлЎң кІӘмқҖ мӨ‘ліө лҚ°мқҙн„° мӮ¬лЎҖмһ…лӢҲлӢӨ.
  • мІ кұ°лҗң кұҙл¬ј(л¶үмқҖ мӢӨм„ )мқҙ мӢ м¶• кұҙл¬ј(л…№мғү лӢӨк°Ғнҳ•)кіј кІ№міҗ мһҲмҠөлӢҲлӢӨ.

4. лі‘н•©лҚ°мқҙн„° мқјм№ҳ м—¬л¶Җ нҷ•мқё

  • л‘җ dataframe AмҷҖ Bмқ„ н•©м№  л•Ң м§‘н•©мқҳ нҸ¬н•ЁкҙҖкі„лҘј мӮҙнҺҙм•ј н•©лӢҲлӢӨ.
  • key feature кё°мӨҖ лқјл©ҙ к°ҖмһҘ мўӢкІ мңјлӮҳ, мқҙл©ҙ Bм—җ кІ°мёЎм№ҳк°Җ мғқкё°кё° л•Ңл¬ёмһ…лӢҲлӢӨ.

  • лҠ” лӢӨмқҢ н•ЁмҲҳлЎң н•ңлІҲм—җ нҷ•мқё к°ҖлҠҘн•©лӢҲлӢӨ.

  • мҳҲм ң DataFrameмқ„ л‘җ к°ң л§Ңл“Өм–ҙм„ң л№„көҗн•ҙ ліҙкІ мҠөлӢҲлӢӨ.

  • м°ЁлЎҖлҢҖлЎң м¶ңл ҘлҗҳлҠ” лҘј ліҖмҲҳлЎң м ҖмһҘн•ҙ нҷңмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.
  • cmp_dfs()н•ЁмҲҳлҠ” key featureл§Ң л№„көҗн•ҳлҜҖлЎң, мўҖ лҚ” мһҗм„ёнһҲ ліј м—¬м§Җк°Җ мһҲмҠөлӢҲлӢӨ.
    • м°Ём§‘н•©мқ„ лІ„лҰҙ кІғмқём§Җ(inner join), кІ°мёЎм№ҳлҘј л§Ңл“Өл©° нҸ¬н•Ён•  кІғмқём§Җ(outer join).
    • көҗм§‘н•©мқҖ key feature мҷё лӢӨлҘё к°’л“ӨмқҖ м–ҙл–Өм§Җ. лҚ°мқҙн„°мқҳ мҲҳ, нғҖмһ…,лӮҙмҡ©мқ„ лӘЁл‘җ нҷ•мқён•©лӢҲлӢӨ.
