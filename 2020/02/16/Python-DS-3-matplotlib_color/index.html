1.1. Matplotlib Color Set

List of Named Colors
Choosing Colormaps in Matplotlib

  • Matplotlibм—җлҠ” нҒ¬кІҢ 3к°Җм§Җ л¶ҖлҘҳмқҳ мғүмғҒмқҙ мһҲмҠөлӢҲлӢӨ.


  • мқҙл“Өмқҙ м–ҙл–»кІҢ м ҖмһҘлҗҳм–ҙ мһҲлҠ”м§ҖлҘј нҷ•мқён•ҳкё° мң„н•ҙ м¶ңл Ҙн•ҙліҙл©ҙ мһ¬л°ҢлҠ” мӮ¬мӢӨмқ„ м•ҢкІҢ лҗ©лӢҲлӢӨ.

    • мӢӨн–үкІ°кіј, к°Ғ н•ӯлӘ©мқҙ лӢҙкі  мһҲлҠ” мғүмғҒм •ліҙмқҳ нҳ•мӢқмқҙ лҜёл¬ҳн•ҳкІҢ лӢӨлҰ…лӢҲлӢӨ.
    • BASE_COLORSлҠ” tuple нҳ•нғңмқё л°ҳл©ҙ лӮҳлЁём§Җ л‘ҳмқҖ stringмһ…лӢҲлӢӨ.
    • к·ёлҰ¬кі  к°ҷмқҖ redлқјкі  н•ҙлҸ„ TABLEAU_COLORSмҷҖ CSS4_COLORSмқҳ к°’, лҢҖмҶҢл¬ёмһҗк°Җ лӢӨлҰ…лӢҲлӢӨ.
    • мӢӨм ң мң„ к·ёлҰјм—җм„ң лҙҗлҸ„ мғүмқҙ лӢӨлҰ…лӢҲлӢӨ.

  • лӢӨмӢң л§җн•ҳл©ҙ Matplotlib лӮҙм—җм„ң лӢӨлЈЁм–ҙм§ҖлҠ” мғүмғҒ н‘ңнҳ„мқҙ н•ң к°Җм§ҖлЎң кі м •лҗҳм§Җ м•Ҡм•ҳкі , мқҙл“ӨмқҖ лӢ№м—°нһҲ matplotlib лӮҙл¶Җ кё°лҠҘмқ„ нҶөн•ҙ ліҖнҷҳмқҙ к°ҖлҠҘн•©лӢҲлӢӨ.

  • Matplotlib лӮҙл¶Җмқҳ colorлҠ” лӢӨмқҢкіј к°ҷмқҙ dictionary нҳ•нғңлЎң н•ңлІҲм—җ мҲҳм§‘н•ҙм„ң нҷңмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • 12к°ңмқҳ м§Ғм„ мқҙ мқјм •н•ң к°„кІ©мқ„ л‘җкі  нҸүн–үн•ҳкІҢ лҠҳм–ҙм„  к·ёлҰјмқёлҚ°, мһҗм„ёнһҲ ліҙл©ҙ мғүмғҒм—җ мӨ‘ліөмқҙ мһҲмҠөлӢҲлӢӨ.

  • мІ«лІҲм§ёмҷҖ л‘җлІҲм§ё н‘ёлҘё м§Ғм„ кіј мҳӨл Ңм§Җмғү м§Ғм„ мқҙ л§Ҳм§Җл§ү л‘җ м§Ғм„ м—җм„ң л°ҳліөлҗҳлҠ” кІғмқ„ нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • matplotlibм—җ кё°ліём ҒмңјлЎң 10к°ңмқҳ мғүмқҙ мЈјм–ҙм§Җкі  м§Ғм„ мқҙ л°ҳліөлҗ  л•Ңл§ҲлӢӨ мқҙлҘј мҲңм°Ём ҒмңјлЎң л°ҳліөн•ҳлҠ” м„Өм •мқҙ лҗҳм–ҙ мһҲкё° л•Ңл¬ёмһ…лӢҲлӢӨ.

  • rcParamsмқҳ axes.prop_cycleн•ӯлӘ©мқ„ нҶөн•ҙ нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • мӢӨн–үкІ°кіј : мғүмғҒ лӘ…лӢЁмқҙ м¶ңл Ҙлҗ©лӢҲлӢӨ.

  • mpl.rc('axes', prop_cycle=)лӘ…л №мқ„ нҶөн•ҙм„ң мҲҳм •н•  мҲҳ мһҲкі ,

  • matplotlibrcнҢҢмқјмқҳ axes.prop_cycle л¶Җл¶„мқ„ нҶөн•ҙ мҲҳм •н•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.
    • мӢӨн–үкІ°кіј : мң„мқҳ к·ёлҰјм—җм„ң м„ мқҳ лӘЁм–‘кіј мғүк№”мқҙ ліҖкІҪлҗҳм—ҲмҠөлӢҲлӢӨ.


  • cyclerлЎң м§Җм •лҗң colorлҠ” вҖңCNвҖқ color selectionмқҙлқјлҠ” к°ңл…җмқ„ нҶөн•ҙ м„ нғқмқҙ к°ҖлҠҘн•©лӢҲлӢӨ.
  • м•„лһҳлҠ” лҸҷмқјн•ң к·ёлһҳн”„м—җ seaborn-whitegrid styleмқҳ мІ«лІҲм§ё(C0)л¶Җн„° л„ӨлІҲм§ё(C4)к№Ңм§Җ colorлҘј лӢӨлҘҙкІҢ м Ғмҡ©н•ҳлҠ” мҳҲмӢңмһ…лӢҲлӢӨ.

2. Color Functions

  • matplotlibм—җ лӮҙмһҘлҗң color functionмқ„ нӣ‘м–ҙлҙ…мӢңлӢӨ.
  • мҳҲм ңлЎң мӮ¬мҡ©н•  color лӢӨм„Ҝ к°ңлҘј лЁјм Җ м •мқҳн•©лӢҲлӢӨ.
  • Lena image мң„м—җ лҚ®мқё мғүмғҒм—җм„ң м•Ң мҲҳ мһҲл“Ҝ alphaлҠ” л¶ҲнҲ¬лӘ…лҸ„лҘј мқҳлҜён•©лӢҲлӢӨ.

2.1. to_hex(), to_rgb(), to_rgba(): мғүмғҒ н‘ңнҳ„ ліҖкІҪ

  • мғүмғҒ мқҙлҰ„мқ„ мһ…л Ҙл°ӣм•„ hex лҳҗлҠ” rgb нҳ•мӢқмңјлЎң ліҖнҷҳн•ҳлҠ” н•ЁмҲҳмһ…лӢҲлӢӨ.
  • л°ұл¬ёмқҙ л¶Ҳм—¬мқјкІ¬мқҙлҜҖлЎң мҡ°м„  pd.DataFrameмңјлЎң мҡ”м•Ҫн•ҙм„ң мӢӨн–үн•ҙліҙкІ мҠөлӢҲлӢӨ.
  • colors DataFrame м¶ңл Ҙ кІ°кіј,
    • to_hex()лҠ” keep_alpha мҳөм…ҳм—җ л”°лқј alpha мұ„л„җмқҙ мӮӯм ң/м¶”к°ҖлҗҳлҠ” л°ҳл©ҙ
    • to_rgb()лҠ” alpha мұ„л„җмқҙ мһҗлҸҷм ҒмңјлЎң мӮӯм ңлҗҳкі ,
    • to_rgba()лҠ” alpha мҳөм…ҳм—җ л”°лқј alpha к°’мқҙ мӣҗлһҳ к°’ vs м§Җм • к°’мңјлЎң м„Өм •лҗ©лӢҲлӢӨ.

2.2. to_rgba_array(): мқјкҙ„ ліҖкІҪ

Numpy Masked Array
Zhang et al., Sensors 14(9) 16128 (2014)

  • 0~1 мӮ¬мқҙ к°’л“Өмқҙ м ҖмһҘлҗң (n, 4) arrayлҘј RGBA arrayлЎң мқјкҙ„ ліҖкІҪн•©лӢҲлӢӨ.

  • (n, 3) arrayк°Җ мһ…л ҘлҗҳлҠ” кІҪмҡ° alpha=1лЎң мһҗлҸҷ м„Өм •лҗҳл©°, column мҲҳк°Җ 3 лҜёл§Ңмқё кІҪмҡ° м—җлҹ¬к°Җ лӮ©лӢҲлӢӨ.

    • мӢӨн–үкІ°кіј: typeмқҙ numpy.ndarrayлЎң л°”лҖҢм—ҲмҠөлӢҲлӢӨ.

  • Numpyм—җлҠ” Masked ArrayлқјлҠ” кё°лҠҘмқҙ мһҲмҠөлӢҲлӢӨ.

  • invalid л“ұ нҠ№м • лҚ°мқҙн„°м—җ maskлҘј м”ҢмӣҢ лі„лҸ„мқҳ мІҳлҰ¬лҘј н•ҳлҠ” кІғмқёлҚ°, to_rgba_array()м—җм„ңлҠ” RGBAлЎң ліҖнҷҳн•ҳм§Җ л§җм•„м•ј н•  л¶Җл¶„мқҙлқјлҠ” мқҳлҜёмһ…лӢҲлӢӨ.
    • мӢӨн–үкІ°кіј: мқјл¶ҖлқјлҸ„ mask мІҳлҰ¬лҗң лҚ°мқҙн„°к°Җ лӘЁл‘җ [0, 0, 0, 0]мңјлЎң ліҖнҷҳлҗҳм—ҲмҠөлӢҲлӢӨ.

2.3. rgb_to_hsv() and hsv_to_rgb()

  • color н‘ңнҳ„ л°©лІ•мқҖ нҒ¬кІҢ RGB(Red, Green, Blue)мҷҖ HSV(Hue, Saturation, Value)лЎң лӮҳлү©лӢҲлӢӨ.
  • BGR(Blue, Green Red), HSL(Hue, Saturation, Lightness), YCrCb(Luma, Blue-difference, Red-difference), CIE(Commission internationale de lвҖҷГ©clairage) л“ұлҸ„ мһҲмңјлӮҳ м—¬кё°м„ңлҠ” лӢӨлЈЁм§Җ м•ҠкІ мҠөлӢҲлӢӨ.
  • RGB color space мң„м—җ HSVлҘј м–№мңјл©ҙ лҢҖлһө мқҙлҹ° кҙҖкі„лҘј к°Җм§‘лӢҲлӢӨ.
  • ліҖнҷҳмӢқмқҙ matplotlibм—җ лӮҙмһҘлҗҳм–ҙ н•ң мӨ„лЎң нҺён•ҳкІҢ мӢӨн–үн•  мҲҳ мһҲмҠөлӢҲлӢӨ.
