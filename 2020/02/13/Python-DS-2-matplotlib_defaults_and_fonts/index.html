matplotlib 3.2.0 #Unicode minus

  • matplotlibмқҖ -(minus) кё°нҳёлҘј н‘ңмӢңн•  л•Ң unicode minus(U+2212)лҘј ASCII hypen(U+002D) ліҙлӢӨ мҡ°м„ м ҒмңјлЎң мӮ¬мҡ©н•©лӢҲлӢӨ.
  • plt.rcParams["axes.unicode_minus"] = False лҘј нҶөн•ҙ ASCII hypenмқ„ мҡ°м„  мӮ¬мҡ©н•ҳлҸ„лЎқ ліҖкІҪн•  мҲҳ мһҲмңјлӮҳ,
  • л•ҢлЎңлҠ” к·ёлҹјм—җлҸ„ л¶Ҳкө¬н•ҳкі  к№Ём ём„ң м¶ңл ҘлҗҳлҠ” кІҪмҡ°л“Өмқҙ мһҲмҠөлӢҲлӢӨ. (ex. -к°Җ м§ҖмҲҳм—җ н‘ңмӢңлҗҳлҠ” кІҪмҡ°)

  • мқҙ мҷём—җлҸ„ non-latin glyphк°Җ м ңлҢҖлЎң н‘ңмӢңлҗҳм§Җ м•ҠлҠ” кІҪмҡ°м—җлҠ” font.family м•һм—җ мғҲ нҸ°нҠёлҘј л„Јм–ҙмЈјлҠ” л°©мӢқмңјлЎң н•ҙкІ°н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • кіөмӢқм ҒмңјлЎң м•Ңл Өм§„ matplotlib н•ңкёҖ мӮ¬мҡ© л°©лІ•мқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  • к°„лӢЁн•ҳкІҢлҠ” мқҙл ҮкІҢл§Ң л„Јм–ҙмӨҳлҸ„ н•ңкёҖмқ„ м¶ңл Ҙн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • мҡҙмҳҒмІҙм ңм—җ кҙҖкі„м—Ҷмқҙ н•ң лҚ©м–ҙлҰ¬мқҳ мҪ”л“ңлҘј мӢӨн–үн•ҙм„ң н•ңкёҖмқ„ кө¬нҳ„н•ҳкі мһҗ н•ңлӢӨл©ҙ лӢӨмқҢ мҪ”л“ң лӯүм№ҳлҘј ліөмӮ¬н•ҳм—¬ мӢӨн–үн•ҳл©ҙ нҺёлҰ¬н•©лӢҲлӢӨ.

  • WindowsмҷҖ Linuxм—җ м Ғмҡ©лҗҳл©°, Linuxм—җм„ң мІҳмқҢ мӢӨн–ү мӢңлҠ” !apt-get м•һм—җ мһҲлҠ” #лҘј м ңкұ°н•ҳкі  мӢӨн–үн•ҳм—¬ нҸ°нҠёлҘј м„Өм№ҳн•ҙм•ј н•©лӢҲлӢӨ.

  • мң„ м„Өм • нӣ„ н•ңкёҖмқҙ л“Өм–ҙк°„ к·ёлһҳн”„лҘј к·ёл ӨліҙкІ мҠөлӢҲлӢӨ.



    • л§Ҳмқҙл„ҲмҠӨлҘј ASCII codeм—җм„ң мӮ¬мҡ©н•ҳкІҢ н–Ҳкё° л•Ңл¬ём—җ yм¶•мқҳ -25лҠ” м •мғҒм ҒмңјлЎң н‘ңмӢңлҗ©лӢҲлӢӨ.
    • к·ёлҹјм—җлҸ„ л¶Ҳкө¬н•ҳкі  unicode л¬ём ңк°Җ л°ңмғқн•ҳм—¬ м§ҖмҲҳмқҳ -к°Җ лІҢл ҲмІҳлҹј(ВӨ: U+00A5)н‘ңмӢңлҗ©лӢҲлӢӨ.

    • superscript minus(вҒ»)лҠ” U+207BмқҙлқјлҠ” лҸ…мһҗ мҪ”л“ңлҘј к°Җм§Җкі  мһҲм§Җл§Ң, matplotlibм—җм„ңлҠ” U+2212лҘј мҳ¬л Өм„ң мӮ¬мҡ©н•ҳкі  мһҲмҠөлӢҲлӢӨ.

    • к°ҖмһҘ к°„нҺён•ң н•ҙкІ°мұ…мқҖ U+2212лҘј к°Җм§Җкі  мһҲлҠ” лӢӨлҘё нҸ°нҠё мӨ‘ н•ҳлӮҳлҘј л¶Ҳлҹ¬мҳӨлҠ” кІғмһ…лӢҲлӢӨ.

    • м—¬лҹ¬к°Җм§ҖлҘј мӢңн—ҳн•ң кІ°кіј лӮҳлҲ”кі л”•кіј к°ҖмһҘ мң мӮ¬н•ң лҢҖмІҙнҸ°нҠёлҠ” Liberation SansмҳҖмҠөлӢҲлӢӨ.
    • к·ё мҷём—җ ubuntu 16.04 кё°ліё нҸ°нҠё мӨ‘м—җм„ңлҠ” Liberation Sans Narrow, Dejavu Sans, UbuntuлҸ„ м“ёл§Ңн•©лӢҲлӢӨ.
    • м•„лһҳ н•ң мӨ„мқ„ мң„ мҪ”л“ңм—җ м¶”к°Җн•ҙм„ң л¬ём ңлҘј н•ҙкІ°н•©лӢҲлӢӨ.
