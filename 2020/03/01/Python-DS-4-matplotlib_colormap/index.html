1. Colormaps

1.1. лҚ°мқҙн„° л¶Ҳлҹ¬мҳӨкё°

Kaggle, Boston Housing
Scikit-Learn, 7.Dataset loading utilities

  • Colormapмқ„ м Ғмҡ©н•  мҳҲм ңлҘј мң„н•ҙ ліҙмҠӨн„ҙ м§‘к°’(boston housing) лҚ°мқҙн„°лҘј л¶Ҳлҹ¬мҳӨкІ мҠөлӢҲлӢӨ.

  • ліҙмҠӨн„ҙ м§‘к°’ лҚ°мқҙн„°лҠ” kaggleм—җм„ң лӢӨмҡҙл°ӣмқ„ мҲҳлҸ„ мһҲм§Җл§Ң нҺёмқҳлҘј мң„н•ҙ scikit-learnм—җ лӮҙмһҘлҗң лҚ°мқҙн„°лҘј л¶Ҳлҹ¬мҳӨкІ мҠөлӢҲлӢӨ.

    • мӢӨн–үкІ°кіј: ліҙмҠӨн„ҙ м§‘к°’мқҳ мқёмһҗ(feature) мқҙлҰ„мқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.
    • к°Ғ мқёмһҗл“Өмқҳ мқҳлҜёлҠ” м—¬кё°м—җм„ң м°ҫмқ„ мҲҳ мһҲм§Җл§Ң, м§ҒкҙҖм Ғмқҙм§Җ м•ҠмҠөлӢҲлӢӨ.
    • мӢӨн–үкІ°кіј: кІ°мёЎм№ҳ м—ҶлҠ” 506к°ңмқҳ лҚ°мқҙн„°м—җ лҢҖн•ң мҡ”м•Ҫмқ„ ліј мҲҳ мһҲмҠөлӢҲлӢӨ.


  • м•һм„  кёҖмқҳ м„Өм •мқ„ мқҙмҡ©н•ҳм—¬ н•ңкёҖ мӮ¬мҡ©мқ„ к°ҖлҠҘн•ҳкІҢ н•ң нӣ„, лҚ°мқҙн„°лҘј мӢңк°Ғнҷ” н•ҙліҙкІ мҠөлӢҲлӢӨ.

  • к°Ғк°Ғмқҳ мқёмһҗл“Өмқҙ м§‘к°’(MEDV)кіј м–ҙл–Ө кҙҖкі„лҘј к°Җм§Җкі  мһҲлҠ”м§Җ м•Ңм•„ліҙкё° мң„н•ҙ мқёмһҗл“Өкіј м§‘к°’кіјмқҳ мғҒкҙҖлҸ„лҘј к·ёл ӨліҙкІ мҠөлӢҲлӢӨ.

    • мӢӨн–үкІ°кіј: л‘җ мқёмһҗ к°„мқҳ кҙҖкі„лҘј м•Ң мҲҳ мһҲлҠ” 14к°ңмқҳ scatter plotмқҙ к·ёл Өм§‘лӢҲлӢӨ.


  • к°Ғ мқёмһҗл“Өкіј м§‘к°’ мӮ¬мқҙмқҳ кҙҖкі„лҠ” м•Ңкё° мү¬мӣҢмЎҢм§Җл§Ң, м •мһ‘ м–ҙл–Ө мқёмһҗл“Өмқём§Җ н•ң лҲҲм—җ л“Өм–ҙмҳӨм§Җ м•ҠмҠөлӢҲлӢӨ.

  • мқҙн•ҙлҸ„лҘј лҸ•кё° мң„н•ҙ xм¶•мқҳ мқёмһҗ лӘ…мқ„ н•ңкёҖлЎң мһ…нҳҖлҙ…мӢңлӢӨ.

  • ax.set_xlabel() м—җ н•ңкёҖ мқҙлҰ„мқ„ л„Јм–ҙмЈјл©ҙ к°ҖлҠҘн•ҳм§Җл§Ң 14к°ң к·ёлһҳн”„м—җ мқјмқјмқҙ л„Јкё° нһҳл“ңлӢҲ мқјкҙ„м ҒмңјлЎң мІҳлҰ¬н•ҙ ліҙкІ мҠөлӢҲлӢӨ.

    • мӢӨн–үкІ°кіј: dictionaryк°Җ мғқм„ұлҗҳм—ҲмҠөлӢҲлӢӨ.

  • set_xlabel()л¶Җл¶„мқ„ лӢӨмқҢкіј к°ҷмқҙ мҲҳм •н•ҳкі  к·ёлҰ¬л©ҙ, н•ңкёҖ к·ёлһҳн”„к°Җ м¶ңл Ҙлҗ©лӢҲлӢӨ.

    • мӢӨн–үкІ°кіј: к·ёлһҳн”„мқҳ к°ҖлҸ…м„ұмқҙ н–ҘмғҒлҗҳм—ҲмҠөлӢҲлӢӨ.

1.2. Colormap Basic

Choosing Colormaps in Matplotlib

  • ліҙмҠӨн„ҙ м§‘к°’ лҚ°мқҙн„°м—җм„ң, DIS(м§Ғм—…м„јн„°к№Ңм§Җ кұ°лҰ¬)мҷҖ MEDV(мЈјнғқ к°ҖкІ©: мӨ‘м•ҷк°’)мқ„ л”°лЎң к·ёл Өлҙ…мӢңлӢӨ.

    • мӢӨн–үкІ°кіј: м§Ғм—…м„јн„°мҷҖмқҳ кұ°лҰ¬мҷҖ мЈјнғқк°ҖкІ© мӮ¬мқҙм—җ м–‘мқҳ мғҒкҙҖкҙҖкі„к°Җ мһҲм–ҙліҙмһ…лӢҲлӢӨ.
    • к·ёлҹ°лҚ°, м—¬кё°м„ңлҠ” 1940л…„ мқҙм „ мЈјнғқ л№„мңЁкіјмқҳ кҙҖкі„к°Җ м „нҳҖ ліҙмқҙм§Җ м•ҠмҠөлӢҲлӢӨ.
    • мң„м—җм„ң 1940л…„ мқҙм „ мЈјнғқ л№„мңЁмқҖ мЈјнғқ к°ҖкІ©кіј л¬ҙкҙҖн•ҙ ліҙмқҙкё° л•Ңл¬ём—җ лҚ” м•Ңкё° м–ҙл өмҠөлӢҲлӢӨ.


  • м„ё ліҖмҲҳ мӮ¬мқҙмқҳ кҙҖкі„лҘј н•ң мһҘмқҳ к·ёлҰјмңјлЎң нҢҢм•…н•ҙ лҙ…мӢңлӢӨ.

  • scatterмқҳ мғүмғҒмқ„ AGE(1940л…„ мқҙм „ мЈјнғқ л№„мңЁ)лЎң мһ…нҳҖліҙкІ мҠөлӢҲлӢӨ.

  • ax.scatter()м—җ cмҷҖ cmap нҢҢлқјлҜён„°лҘј м¶”к°Җн•ҙмЈјл©ҙ лҗ©лӢҲлӢӨ.

    • c: color. мғүмғҒ мқҙлҰ„мқҙлӮҳ мҪ”л“ңлҘј мһ…л Ҙн•  мҲҳлҸ„ мһҲм§Җл§Ң, ліҖмҲҳлӘ…мқ„ мһ…л Ҙн•  мҲҳ мһҲмҠөлӢҲлӢӨ.
    • cmap: colormap. м „мІҙ лӘ©лЎқмқҖ м—¬кё°м—җ мһҲмҠөлӢҲлӢӨ.
    • мӢӨн–үкІ°кіј: scatterм—җ AGE(1940л…„ мқҙм „ мЈјнғқ л№„мңЁ) лҚ°мқҙн„°к°Җ мһ…нҳҖм§‘лӢҲлӢӨ.
    • DIS(м§Ғм—…м„јн„°к№Ңм§Җ кұ°лҰ¬)к°Җ м§§мқ„ мҲҳлЎқ л…ёнӣ„н•ң мЈјнғқмқҙ л§ҺлӢӨлҠ” кІғмқ„ м•Ң мҲҳ мһҲмҠөлӢҲлӢӨ.

  • Matplotlibм—җм„ң лӢӨм–‘н•ң colormapмқ„ м§Җмӣҗн•©лӢҲлӢӨ.

  • нҷҲнҺҳмқҙм§Җм—җм„ң мқҙлҰ„кіј мғүмғҒ л°°м—ҙмқ„ ліҙкі  мӣҗн•ҳлҠ” кІғмқ„ кіЁлқј м Ғмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

1.3. Colorbar

matplotlib colorbar

  • colormapмңјлЎң лӮҳнғҖлӮё лҚ°мқҙн„°м—җм„ң, м–ҙл–Ө мғүмғҒмқҙ м–ҙл–Ө к°’м—җ н•ҙлӢ№н•ҳлҠ”м§Җ лӘ…мӢңн•ҙм•ј н•  н•„мҡ”к°Җ мһҲмҠөлӢҲлӢӨ.

  • мқҙлҹҙ л•Ң к°ҖмһҘ ліҙнҺём ҒмңјлЎң мӮ¬мҡ©лҗҳлҠ” кІғмқҙ colorbar мһ…лӢҲлӢӨ.

    • мӢӨн–үкІ°кіј: colorbarк°Җ мғқм„ұлҗҳм—ҲмҠөлӢҲлӢӨ.

  • colorbarлҘј лі„лҸ„мқҳ к°қмІҙлЎң м§Җм •н•ҳкі  set_label()мқ„ мқҙмҡ©н•ҙ labelмқ„ м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • мӢӨн–үкІ°кіј: colorbarм—җ labelмқҙ мғқм„ұлҗҳм—ҲмҠөлӢҲлӢӨ.

  • colobarмқҳ мҶҚм„ұмқ„ лӢӨмқҢкіј к°ҷмқҖ м„ё к°Җм§Җ л°©мӢқмңјлЎң м ңм–ҙн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  1. axes properties: colorbarк°Җ м Ғмҡ©лҗҳлҠ” axesм—җ лҢҖн•ң мҶҚм„ұ.
  • orientation (horizontal or vertical)
  • fraction (default: 0.15. colorbarк°Җ м°Ём§Җн•ҳлҠ” мҳҒм—ӯмқҳ л№„мңЁ)
  • aspect (default: 20. colorbarмқҳ кёҙ ліҖ : м§§мқҖ ліҖ л№„мңЁ)
    л“ұмқҙ мһҲмҠөлӢҲлӢӨ.
  • мӢӨн–үкІ°кіј: colorbarк°Җ к°ҖлЎңлЎң, лҡұлҡұн•ҳкІҢ, к·ёлһҳн”„мқҳ м Ҳл°ҳмқ„ м°Ём§Җн•ҳкІҢ лҗҳм—ҲмҠөлӢҲлӢӨ.
  • мқҙлҜём§ҖлҘј нҒҙлҰӯн•ҳкұ°лӮҳ л“ңлһҳк·ён•ҳмӢңл©ҙ л°°кІҪм—җ к°Җл Өм§„ мқҙлҜём§Җмқҳ м Ҳл°ҳмқ„ ліҙмӢӨ мҲҳ мһҲмҠөлӢҲлӢӨ.
  1. colorbar properties: colorbar мһҗмІҙмқҳ мҶҚм„ұ.
  • ticks (default: automatic)
  • format (tick values нҳ•мӢқ)
  • drawedges (default: False. color кө¬к°„л§ҲлӢӨ мңӨкіҪм„  н‘ңмӢң)
    л“ұмқҙ мһҲмҠөлӢҲлӢӨ.

  • мӢӨн–үкІ°кіј: м§Җм •лҗң мң„м№ҳм—җл§Ң, м§ҖмҲҳ нҳ•мӢқмңјлЎң, drawedgeмқҳ False(0)кіј True(1)к°Җ мғқм„ұлҗҳм—ҲмҠөлӢҲлӢӨ.

  • drawedges мҳөм…ҳ м Ғмҡ©мӢң мғүмғҒмқҙ м–ҙл‘җмӣҢм§ҖлҠ”лҚ°, 1 лӢЁмң„лЎң edgeк°Җ мғқм„ұлҗҳкё° л•Ңл¬ёмһ…лӢҲлӢӨ.

  • кҙҖл Ёлҗң мқҙмҠҲмҷҖ workaroundк°Җ мқҙкіім—җ м •лҰ¬лҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

  • мң„мҷҖ к°ҷмқҙ м—¬лҹ¬ к°ңмқҳ subplotмқ„ к°Җм§ҖлҠ” кІҪмҡ° кіөнҶө colorbarлҘј мӮҪмһ…н•  кІҪмҡ°к°Җ мһҲмҠөлӢҲлӢӨ.

  • к·ёлҹ¬лӮҳ м§ҒкҙҖм ҒмңјлЎң plt.colorbar()лҘј л§Ё л§Ҳм§Җл§үмңјлЎң мҳ®кё°кі  plt.tight_layout()мқ„ мӢӨн–үн•ҳл©ҙ plotкіј colorbarк°Җ кІ№м№ҳкІҢ лҗ©лӢҲлӢӨ.

  • л”°лқјм„ң, plt.tight_layout()м—җ rect мҳөм…ҳмқ„ мЈјм–ҙ plot мҳҒм—ӯмқ„ м§Җм •н•ҙмӨҳм•ј н•©лӢҲлӢӨ.

  • rect мҳөм…ҳмқҖ к·ёлҰјмқҳ м–ҙлҠҗ л¶Җл¶„м—җ axesк°Җ л“Өм–ҙк°ҖлҠ”м§ҖлҘј tuple нҳ•нғңлЎң, (left, bottom, right, top)мңјлЎң м§Җм •н•©лӢҲлӢӨ. кё°ліёк°’мқҖ (0, 0, 1, 1)мһ…лӢҲлӢӨ.

Matplotlib, Changes to the default style

  • лӢӨлҘё кёҖм—җм„ң лӢӨлЈЁм—Ҳл“Ҝ мқҙлҜём§Җк°Җ лӢЁмҲң к·ёлҰјмқҙ м•„лӢҲлқј x, y 2м°Ёмӣҗ кіөк°„мғҒмқҳ лҚ°мқҙн„°мқј мҲҳ мһҲмҠөлӢҲлӢӨ.

  • matplotlibм—җм„ң 2D DataлЎңм„ңмқҳ ImageлҘј мқҪм–ҙл“Өмқё нӣ„ LUTлҘј ліҖкІҪн•ҳлҠ” л°©лІ•мқ„ м •лҰ¬н•©лӢҲлӢӨ.

  • .tif мқҙлҜём§ҖлҘј мқҪкі  м ҖмһҘн•ҳкё° мң„н•ҙ Pillowк°Җ н•„мҡ”н•©лӢҲлӢӨ.

  • мҳҲм ң мқҙлҜём§ҖлҠ” м—¬кё°м—җм„ң лӢӨмҡҙлЎңл“ң к°ҖлҠҘн•©лӢҲлӢӨ.

    • мӢӨн–үкІ°кіј: кё°ліё colormapмқё viridisк°Җ м Ғмҡ©лҗң мқҙлҜём§Җк°Җ мғқм„ұлҗ©лӢҲлӢӨ.

  • plt.imshow()м—җ cmap= мқёмһҗлҘј м§Җм •н•ҳл©ҙ мқҙлҜём§Җм—җ м Ғмҡ©лҗҳлҠ” colormapмқ„ ліҖкІҪн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • нҢҢмқјлЎң м ҖмһҘн•  л•ҢлҠ” лі„лҸ„лЎң cmap= мқёмһҗлҘј м§Җм •н•ҙм•ј н•©лӢҲлӢӨ.

  • к·ёл Үм§Җ м•Ҡмңјл©ҙ нҷ”л©ҙкіјлҠ” л¬ҙкҙҖн•ң кё°ліё viridis colormapмқҙ м Ғмҡ©лҗң кІ°кіјлҘј ліј мҲҳ мһҲмҠөлӢҲлӢӨ.

    • мӢӨн–үкІ°кіј: colormapмқҙ oceanмңјлЎң ліҖкІҪлҗҳм—ҲмҠөлӢҲлӢӨ.
