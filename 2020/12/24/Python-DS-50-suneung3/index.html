6. м„ұл№„ кө¬н•ҳкё°

6.1. м—¬м„ұ м§Җмӣҗмһҗ 1мқёлӢ№ лӮЁм„ұ м§Җмӣҗмһҗ мҲҳ

  • мқјл°ҳм ҒмңјлЎң м„ұл№„лҘј м—¬м„ұ 1мқёлӢ№ лӮЁм„ұмқҳ мҲҳлЎң м •мқҳн•©лӢҲлӢӨ.

  • мқ‘мӢңмңЁкіј к°ҷмқҖ мҡ”л №мңјлЎң м„ұл№„лҘј л„Јм–ҙлҙ…мӢңлӢӨ.

  • мқ‘мӢңмңЁмқ„ л„ЈлҚҳ for loopм—җ м„ұл№„лҘј к°ҷмқҙ л„ЈмҠөлӢҲлӢӨ.

  • legendлҸ„ м¶”к°Җн•©лӢҲлӢӨ.

6.2. лӮЁм„ұ м§Җмӣҗмһҗ 1мқёлӢ№ м—¬м„ұ м§Җмӣҗмһҗ мҲҳ

  • м—¬м„ұ мһ¬мҲҳмғқ л№„мңЁмқҙ лҠҳм–ҙлӮҳлҠ” кІғмқём§ҖлҠ” нҳ„мһ¬лЎңм„ңлҠ” м•Ң мҲҳ м—Ҷм§Җл§Ң, к·ёл ҮлӢӨкі  н•ҙлҸ„ мһ¬мҲҳлҘј н—Ҳмҡ©н•ҳлҠ” л№„мңЁмқҙ мҰқк°Җн•ңлӢӨкі лҸ„ ліј мҲҳ мһҲмқ„ н…ҢлӢҲк№Ңмҡ”.
