0.0. ImageJ Cookbook

  • 코딩에서 Cookbook이란, 제목처럼 실제로 사용할 수 있는 코드나 간단한 기술의 모음입니다.


  • 음식을 배울 때 재료의 특성과 칼질의 기초부터 배울 수도 있겠지만 주말의 허기를 달래는데 요리사 자격증까지 필요하진 않습니다.

  • 인터넷 블로그와 유튜브에 널린 레시피처럼, ImageJ의 GUI 환경을 이용하거나 ImageJ의 스크립트 창을 열고 python 코드 몇 줄을 Ctrl+C/V 후 살짝 고쳐 원하는 결과를 얻을 수 있는 코드를 모아두려 합니다.

  • GUI 활용은 컴퓨터를 기본적인 수준으로 다룰 수 있는 누구나 가능한 수준으로 레시피를 제공하고자 합니다.

  • Script는 GUI에 비해 어렵게 느껴지실 수 있으나, ImageJ Tutorial의 5. Python Script 작성까지 읽어보신 분들은 아래 코드를 사용하실 수 있으리라 생각합니다.

  • 좋은 ImageJ Cookbook을 발견하시면 이메일로 제보 부탁드립니다. 아래 레퍼런스에 추가하고 유용한 내용은 포스팅으로 추가해서 공유하겠습니다.

0.1. ImageJ Cookbook Installation

  • ImageJ에서 자체적으로 제공하는 Cookbook 이 있습니다.
  • 예를 들면 Particle Analysis 는 어떤 절차를 거쳐 진행되는지 예시를 보여주고 ImageJ에 원클릭으로 실행할 수 있는 메뉴를 제공합니다.
  • 파라미터와 절차가 본인의 이미지에 맞게 설정된 것이 아니므로 원클릭 실행에는 큰 기대를 하기 힘들지만 절차를 보고 공부하기는 좋습니다.
  • 공식 홈페이지에 실린 내용을 소개드립니다.

  1. ImageJ를 설치합니다.

  2. 업데이트를 위해 Help > Update... 로 들어갑니다.



  3. 좌측 하단 Manage update sites에서 Cookbook을 선택하고, Close를 누릅니다.

    • 어떤게 있나 살펴보셔도 좋습니다.
    • Tensorflow와 TEM suite라는 것도 있습니다.


  4. 설치되는 동안 잠시 기다립니다.



  5. ImageJ를 재시작하면 Cookbook 탭에 연결된 기능들을 볼 수 있습니다.



  6. 홈페이지 에서 제공하는 설명과 내 이미지에 실행한 결과를 비교하며 공부합니다.

    • Particle Analysis 예제 페이지입니다.
