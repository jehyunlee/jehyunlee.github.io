1. Calibrationмқҳ мқҳлҜё

  • мқҙлҜём§Җ нҢҢмқјмқҳ кө¬мЎ°м—җ лҢҖн•ң кёҖм—җм„ң, мқҙлҜём§ҖлҠ” 2м°Ёмӣҗ кіөк°„мғҒм—җ мң„м№ҳн•ң pixelмқҳ м§‘н•©мқҙлқјкі  л§җм”Җл“ңл ёмҠөлӢҲлӢӨ.

  • лҢҖк°ң pixel н•ҳлӮҳлӢ№ нқ‘л°ұ(Grayscale) мқҙлҜём§Җмқҳ кІҪмҡ° мҲ«мһҗ н•ҳлӮҳк°Җ, м»¬лҹ¬ мқҙлҜём§Җмқҳ кІҪмҡ° мҲ«мһҗ 3к°ңк°Җ RGB channelмқҳ нҳ•нғңлЎң н• лӢ№лҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

  • к·ёлҹ¬лӮҳ мқҙлҜём§Җм—җ м°ҚнһҢ лҢҖмғҒмқҳ мӢӨм ң нҒ¬кё°лҠ” pixelкіј лӢӨлҘё, meter л“ұмқҳ лӢЁмң„лЎң мқҙлЈЁм–ҙм ё мһҲмҠөлӢҲлӢӨ. мқҙлҘј ліҙм •н•ҳлҠ” кіјм •мқ„ calibrationмқҙлқјкі  н•ҳл©°, scaleмқ„ pixelлӢЁмң„ кёёмқҙлӮҳ л„“мқҙм—җ кіұн•©лӢҲлӢӨ.

    • ImageJм—җм„ң мқҙлҜём§ҖлҘј м—ҙкі  нҸ¬мқён„°лҘј к·ёлҰј мң„м—җ мҳ¬лҰ¬л©ҙ, л©”мқё м°Ҫм—җ x, yмўҢн‘ңмҷҖ н•Ёк»ҳ pixelк°’мқҙ лӮҳмҳөлӢҲлӢӨ.
    • к·ёлҹ¬лӮҳ м–ҙл–Ө мқҙлҜём§ҖлҠ” ImageJ л©”мқё м°Ҫмқҳ x, yмўҢн‘ң, pixelк°’м—җ кҙ„нҳё()к°Җ л¶ҷм–ҙмһҲмҠөлӢҲлӢӨ.
    • м•„лһҳ мҳҲм ңмқҳ мӣҗліёмқҖ м—¬кё°м—җм„ң лӢӨмҡҙлЎңл“ң л°ӣмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.
    • кҙ„нҳё м•Ҳмқҳ к°’мқҖ pixelлӢЁмң„, кҙ„нҳё л°–мқҳ к°’мқҖ ліҙм •лҗң(calibrated) мӢӨм ң кёёмқҙмһ…лӢҲлӢӨ.
    • мқҙлҜём§Җ мҡ°мёЎ н•ҳлӢЁ $ 2\ \mu\text{m} $ scale barмқҳ кёёмқҙлҠ” 484 px мһ…лӢҲлӢӨ.
    • л©”нғҖлҚ°мқҙн„°лЎңл¶Җн„° м–»мқҖ scaleмқҙ 4.13 nm/pxмқҙлҜҖлЎң, нҷ•мқёмқ„ мң„н•ҙ кіұн•ҙліҙл©ҙ $ 484 \times 4.13 = 1998.92\ \text{nm} $ мқҙлҜҖлЎң $ \approx 2 \mu\text{m} $ мһ…лӢҲлӢӨ.

  • ImageJ CalibrationмқҖ нҒ¬кІҢ л‘җ к°Җм§ҖлЎң лӮҳлүҳм–ҙм§‘лӢҲлӢӨ.
    • Spatial Calibration : x, yл“ұ кёёмқҙ л°©н–Ҙмқ„ ліҙм •н•©лӢҲлӢӨ.
    • Density Calibration: pixelк°’мқ„ ліҙм •н•©лӢҲлӢӨ.

2. Calibrationмқҳ нҷңмҡ©

  • Calibrationмқҙ лҗң мқҙлҜём§ҖлҘј мқҙмҡ©н•ҙ мёЎм •н•ҙлҙ…мӢңлӢӨ.

    • DEM: Digital Elevation ModelмқҙлқјлҠ” мў…лҘҳмқҳ лҚ°мқҙн„°к°Җ мһҲмҠөлӢҲлӢӨ.
    • мқҙлҜём§Җ нҳ•мӢқмқҙм§Җл§Ң pixelм—җ лӢҙкёҙ к°’мқҖ н•ҙлӢ№ мң„м№ҳмқҳ кі лҸ„лҘј мқҳлҜён•©лӢҲлӢӨ.
    • м ңн•ңлҗң к°’(8 bit Grayscale мқҙлҜём§Җ: 0~255) к°’л§ҢмңјлЎң нӣЁм”¬ л„“мқҖ лІ”мң„мқҳ кі лҸ„лҘј нҢҢм•…н•ҙм•ј н•ҳлҜҖлЎң pixelк°’м—җ scaleмқҙ м Ғмҡ©лҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

  • м—¬кё°м—җм„ң мқҙлҜём§ҖлҘј лӢӨмҡҙлЎңл“ңл°ӣм•„ ImageJм—җм„ң м—ҪлӢҲлӢӨ.

    • Image > Lookup Tablesм—җм„ң Green Blue FireлҘј м„ нғқн•©лӢҲлӢӨ.

    • нқ”нһҲ LUTлқјкі  л§җн•ҳлҠ” Loopup TableмқҖ Grayscaleм—җ лӢҙкёҙ Scalar к°’мқ„ мӢңк°Ғнҷ”н•  colormapмһ…лӢҲлӢӨ.

    • мӢӨм ң к°’мқҖ ліҖкІҪн•ҳм§Җ м•Ҡкі  ImageJм—җм„ң ліҙм—¬мЈјлҠ” мӢңк°Ғм Ғмқё нҡЁкіјл§Ң ліҖкІҪлҗ©лӢҲлӢӨ.

    • Analyze > Surface Plotмқ„ м„ нғқн•ҳкі  мҳөм…ҳм—җм„ң Draw Wireframe, Shade, Draw Axis, SmoothлҘј м„ нғқн•©лӢҲлӢӨ. 2м°Ёмӣҗ к·ёлҰјмқҙ 3м°ЁмӣҗмңјлЎң ліҖнҷҳлҗ©лӢҲлӢӨ. z м¶• мҲ«мһҗлҘј ліҙл©ҙ 167 ~ 2408лЎң н‘ңмӢңлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ. Calibrationмқҙ м Ғмҡ©лҗң кІғмһ…лӢҲлӢӨ.

    • Analyze > 3D Surface Plotмқ„ м„ нғқн•ҙлҸ„ л№„мҠ·н•ң к·ёлҰјмқ„ м–»мқ„ мҲҳ мһҲмҠөлӢҲлӢӨ. 3D нҡҢм „мқ„ л№„лЎҜн•ҙм„ң Grid Size, Smoothing л“ұмқ„ мӢӨмӢңк°„мңјлЎң мЎ°м •н•  мҲҳ мһҲлҠ” interactive plotмһ…лӢҲлӢӨ. м—¬кё°м„ңлҠ” z м¶•м—җ Calibrationмқҙ м Ғмҡ©лҗҳм§Җ м•Ҡм•ҳмҠөлӢҲлӢӨ.

3.Calibration (GUI)

3.1. Spatial Calibration (GUI)

  • мқҙлҜём§Җмқҳ Scale BarлҘј мқҙмҡ©н•ҳм—¬ Spatial Calibrationмқ„ м„Өм •н•ҙ ліҙкІ мҠөлӢҲлӢӨ.

    • л©”мқё м°Ҫм—җм„ң Line Selection Toolмқ„ м„ нғқн•ҳкі  Scale BarлҘј л”°лқј м„ мқ„ кёӢмҠөлӢҲлӢӨ. мІ« м§Җм җмқ„ нҒҙлҰӯн•ҳкі  shiftнӮӨлҘј лҲ„лҘё мұ„ л§Ҳмҡ°мҠӨлҘј мқҙлҸҷн•ҳл©ҙ нҸ¬мқён„°лҘј мҲҳнҸүмңјлЎңл§Ң мӣҖм§Ғмқј мҲҳ мһҲмҠөлӢҲлӢӨ.

    • Calibrationмқ„ м Ғмҡ©н•  мқҙлҜём§ҖлҘј м—¬кё°м—җм„ң лӢӨмҡҙлЎңл“ңл°ӣм•„ ImageJм—җм„ң м—ҙкі , Analyze > Set Scale...мқ„ м„ нғқн•ҳм—¬ Scale м„Өм • м°Ҫмқ„ м—ҪлӢҲлӢӨ.

    • мЎ°кёҲ м „ мҶҗмңјлЎң к·ём—ҲлҚҳ м„ мқҳ кёёмқҙк°Җ Distance in pixelsлЎң 483мқҙ лӮҳмҳөлӢҲлӢӨ.

    • Scale Barм—җ $2\ \mu \text{m} = 2000\ \text{nm}$ лқјкі  лҗҳм–ҙ мһҲмңјлҜҖлЎң Known distanceм—җ 2000мқ„ мһ…л Ҙн•ҳкі  Unit of lengthм—җ nmлҘј мһ…л Ҙн•©лӢҲлӢӨ.

    • ліё мқҙлҜём§Җм—җл§Ң м Ғмҡ©н•  CalibrationмқҙлҜҖлЎң GlobalмқҖ мІҙнҒ¬лҗҳм§Җ м•ҠмқҖ мұ„лЎң лҶ”л‘” мұ„ OKлҘј лҲ„лҘҙл©ҙ Calibrationмқҙ м Ғмҡ©лҗ©лӢҲлӢӨ.

  • л°©кёҲ м„Өм •н•ң Calibrationмқ„ мқҙмҡ©н•ҳм—¬ Particleмқҳ м§ҖлҰ„мқ„ мёЎм •н•ҙ ліҙкІ мҠөлӢҲлӢӨ.

    • м§ҖлҰ„мқ„ мһҙ Particleмқ„ м •н•ҳкі , Line Selection Toolмқ„ мқҙмҡ©н•ҙм„ң м§ҖлҰ„м—җ л§һм¶° м„ мқ„ кёӢмҠөлӢҲлӢӨ.

    • Analyze > Set Measurementsм—җм„ң мёЎм •н•  н•ӯлӘ©мқ„ м„ нғқн•©лӢҲлӢӨ. Angleкіј LengthлҠ” кё°ліёмңјлЎң м¶ңл ҘлҗҳлӢҲ м¶”к°ҖлЎң м–»мқ„ к°’л“Өмқ„ м„ нғқн•ҳкі , н•„мҡ” м—ҶлҠ” к°’л“Өмқ„ м„ нғқ н•ҙм ңн•ҳл©ҙ лҗ©лӢҲлӢӨ.

    • Analyze > MeasureлҘј м„ нғқн•ҳл©ҙ Results м°Ҫмқҙ лңЁкі  Length = 896.401мқҙ м¶ңл Ҙлҗ©лӢҲлӢӨ. Results м°Ҫм—җм„ң File > Save as...лҘј м„ нғқн•ҳл©ҙ .csvнҢҢмқјлЎң м ҖмһҘн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • Analyze > Plot Profileмқ„ м„ нғқн•ҳл©ҙ Particleмқҳ м§ҖлҰ„мқ„ к°ҖлЎңм§ҖлҘҙлҠ” м„ мқ„ л”°лқј Intensity Profileмқ„ м–»мқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

  • ImageJм—җм„ң мқҙлҜём§Җм—җ Scale BarлҘј мӮҪмһ…н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • Analyze > Tools > Scale Bar...лҘј м„ нғқн•©лӢҲлӢӨ.

    • Scale Bar м°Ҫм—җм„ң Calibration лӢЁмң„(nm) кё°мӨҖмқҳ кёёмқҙмҷҖ Scale Barмқҳ Height, Font Size, Location л“ұмқ„ м„Өм •н•©лӢҲлӢӨ. м„Өм •к°’м—җ л”°лқј мқҙлҜём§Җ мң„м—җ мӢӨмӢңк°„мңјлЎң Scale Barк°Җ мғқм„ұлҗҳлӢҲ ліҙмӢңл©ҙм„ң мӣҗн•ҳлҠ” мғҒнғңлЎң м„Өм •н•ҳл©ҙ лҗ©лӢҲлӢӨ.

    • $2000\ \text{nm}$ лҢҖмӢ  $2\ \mu \text{m}$лЎң м¶ңл Ҙлҗҳкё°лҘј мӣҗн•ҳмӢңл©ҙ, calibrationмқ„ $\mu \text{m}$ лӢЁмң„лЎң м§„н–үн•ҳмӢңл©ҙ лҗ©лӢҲлӢӨ.

3.2. Density Calibration (GUI)

  • Density Calibrationмқ„ мң„н•ҙм„  лӢӨмқҢ л‘җ к°Җм§Җ мӮ¬н•ӯмқҙ н•„мҡ”н•©лӢҲлӢӨ.
    1. л‘җ м§Җм җ мқҙмғҒмқҳ pixelкіј лҚ°мқҙн„°(value) мҢҚ
    2. Density Calibrationмқҳ н•ЁмҲҳ(function): Linear? Polynomial? Exponential?

  • Density Calibrationмқ„ мң„н•ң лҚ°мқҙн„°к°Җ мЈјм–ҙм§Җл©ҙ мўӢм§Җл§Ң, к·ёл Үм§Җ м•ҠмқҖ кІҪмҡ° мқҙлҜём§Җ н•ңмј мқҳ лІ”лЎҖ(legend)лҘј мқҙмҡ©н•ҙм„ңлқјлҸ„ мһ…л Ҙн•ҙм•ј н•©лӢҲлӢӨ.

  • GeoTIFF мқҙлҜём§Җ лҚ°мқҙн„°лҘј мқҙмҡ©н•ҙм„ң Density Calibrationмқ„ н•ҙ ліҙкІ мҠөлӢҲлӢӨ.

    • GeoTIFF нҳ•мӢқмқҖ м§ҖлҰ¬м •ліҙ(GIS)мҷҖ кё°мғҒ л¶„м•јм—җ мЈјлЎң мӮ¬мҡ©лҗҳл©°, мқҙлҜём§Җ мҷём—җлҸ„ map projection, coordinate systems, ellipsoids л“ұмқҳ лҚ°мқҙн„°лҘј н•Ёк»ҳ лӢҙкі  мһҲмҠөлӢҲлӢӨ.
    • м—¬кё°м—җм„ң мҳҲм ңнҢҢмқјмқ„ лӢӨмҡҙл°ӣмҠөлӢҲлӢӨ.
    • ImageJм—җм„ң нҢҢмқјмқ„ м—ҙкі , Analyze > CalibrateлҘј м„ нғқн•©лӢҲлӢӨ.
    • Calibrate м°Ҫм—җм„ң, мҷјмӘҪм—җ pixel к°’мқ„ л„Јкі  мҳӨлҘёмӘҪмқҖ мқҙм—җ н•ҙлӢ№н•ҳлҠ” valueлҘј л„Јм–ҙмӨҚлӢҲлӢӨ.
      1. мҷјмӘҪ кіөк°„м—җ 0кіј 254лҘј л„ЈмҠөлӢҲлӢӨ. лҚ°мқҙн„° мӮ¬мқҙлҠ” EnterнӮӨлЎң лқ„мӣҢмӨҚлӢҲлӢӨ.
      2. мҳӨлҘёмӘҪ кіөк°„м—җ -2мҷҖ 45лҘј л„ЈмҠөлӢҲлӢӨ. м—ӯмӢң EnterнӮӨлЎң л¶„лҰ¬н•ҙ мӨҚлӢҲлӢӨ.
      3. м„ӯм”Ё мҳЁлҸ„ лҚ°мқҙн„°мһ…лӢҲлӢӨ. unitм—җ н•ҙлӢ№н•ҳлҠ” кіім—җ Degrees CelsiusлҘј л„ЈмҠөлӢҲлӢӨ.
      4. 1м°ЁмӢқ лҚ°мқҙн„°мқҙлҜҖлЎң Function л©”лүҙм—җм„ң Straight Lineмқ„ м„ нғқн•©лӢҲлӢӨ.
      5. OKлҘј лҲ„лҘҙл©ҙ Calibration Functionмқҙ ліҙмқј кІғмһ…лӢҲлӢӨ.
      6. Calibration Functionмқ„ лӢ«кі  мқҙлҜём§Җ мң„м—җ л§Ҳмҡ°мҠӨлҘј к°Җм ёк°Җл©ҙ, л©”мқё м°Ҫм—җ нҸ¬мқён„°к°Җ мң„м№ҳн•ң м§Җм җмқҳ x, y мўҢн‘ңмҷҖ н•Ёк»ҳ valueк°Җ ліҙмһ…лӢҲлӢӨ.
      7. мқҙлҜём§ҖлҘј File > Save Asм—җм„ң .tifнҳ•мӢқмңјлЎң м ҖмһҘн•ҳл©ҙ Calibrationмқҙ н•Ёк»ҳ м ҖмһҘлҗ©лӢҲлӢӨ.

  • Density Calibrationмқ„ мқҙмҡ©н•ҙ мқҙлҜём§Җ лҚ°мқҙн„° л¶„м„қмқ„ н•ҙ ліҙкІ мҠөлӢҲлӢӨ.

    • Analysis > Set Measurementsм—җм„ң Area, Mean gray value, Standard deviation, Modal gray value, Min & Max gray valueлҘј м„ нғқн•ҳкі  OKлҘј лҲҢлҹ¬ м°Ҫмқ„ лӢ«мҠөлӢҲлӢӨ.
    • Analysis > MeasurementлҘј нҒҙлҰӯн•ҳл©ҙ Resultsм°Ҫмқҙ лң№лӢҲлӢӨ.
    • Set Measuresм—җм„ң м§Җм •н•ң л¶„м„қ кІ°кіјк°Җ лӢҙкІЁ мһҲмҠөлӢҲлӢӨ.

  • Scale BarмІҳлҹј Density Calibrationмқ„ LegendлЎң мӮҪмһ…н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • Analyze > Tools > Calibration Bar...лҘј м„ нғқн•©лӢҲлӢӨ.

    • Scale BarмҷҖ лҢҖмІҙлЎң л№„мҠ·н•ҳм§Җл§Ң Number of Labels, Decimal Places, Zoom Factorл“ұ лӢӨлҘё мқёмһҗл“Өмқҙ мһҲмҠөлӢҲлӢӨ.

    • м§Ғм ‘ мқёмһҗлҘј л°”кҝ”к°Җл©ҙм„ң Legendк°Җ мӢӨмӢңк°„мңјлЎң м–ҙл–»кІҢ ліҖн•ҳлҠ”м§Җ мӮҙнҺҙліҙмӢңкё° л°”лһҚлӢҲлӢӨ.
